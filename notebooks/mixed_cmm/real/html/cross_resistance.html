Cross-resistance across drugs (original dataset)¶

Bottom line. The clearest cross-resistance is loss of F420 activation raising both delamanid and pretomanid; the efflux pump raises both bedaquiline and clofazimine (the signal is spread across several efflux variants). Linezolid stays clean.

Idea. Put several drug MICs in one graph and forbid MIC <-> MIC edges. Then two drugs can only become correlated through a shared mutation/burden parent, so a feature pointing into more than one MIC is the cross-resistance driver.

Linezolid (a separate ribosomal mechanism, no resistant isolates here) is kept only in the joint graph as a negative control: the efflux/F420 drivers should not point to it.

Resistance type is deliberately excluded from these graphs: XDR is defined by bedaquiline/linezolid resistance, so type is a deterministic descendant (collider) of those MICs, and conditioning on it would leak the outcome.

How to read it. An arrow X -> MIC means X raises that MIC even after accounting for everything else; the number on the arrow is how often the link survives when we refit on random 80% subsamples (higher = more reliable; >= 0.5 is treated as solid).

Node colours: salmon = drug MIC (outcome), blue = mutation, orange = rare-variant pathway burden, grey = lineage, khaki = resistance type.

What the burdens are, and how they were pooled¶

Most resistance variants are individually too rare (< 5% of strains) to estimate on their own. So within each known resistance mechanism, the rare non-synonymous variants are pooled into one binary indicator (the burden): it is 1 for a strain if that strain carries any such variant in the mechanism's genes, else 0. Common variants are kept as individual nodes; synonymous (silent) changes are dropped. This trades 'which exact variant' for 'is the mechanism hit', which is what carries the signal when each variant alone is unanalysable.

  • F420 activation (genes ddn, fgd1, fbiA, fbiB, fbiC, fbiD). Delamanid and pretomanid are pro-drugs: they must be activated inside the bacterium by the Ddn nitroreductase, which needs the cofactor F420 (produced by Fgd1 and the FbiA-D enzymes). A loss-of-function anywhere in this pathway means the pro-drug is never activated, so the strain resists both nitroimidazoles. burden_f420_activation = carries any rare variant in these genes.
  • Efflux (genes rv0678, mmpL5, mmpS5, pepQ, rv1979c). The MmpS5-MmpL5 pump exports bedaquiline and clofazimine; Rv0678 normally represses it. Mutations that break Rv0678 (or alter the pump) increase efflux and raise the MIC of both drugs. burden_efflux = carries any rare variant in these genes.

Reading it¶

  • F420 burden -> delamanid is the cleanest cross-resistance signal and reaches pretomanid: a pooled activation defect raises both nitroimidazole MICs.
  • Efflux is real but split: the same mmpl5/rv0678/rv1979c variants point into both bedaquiline and clofazimine, but co-occurrence splits the credit so none clears 0.5.
  • The joint graph is underpowered; the focused graphs concentrate the signal. Linezolid stays clean, as it should.
